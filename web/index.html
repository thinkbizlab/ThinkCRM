    How the Leaderboard Score Works

    Each person's score reflects their progress toward monthly KPI targets.

    Score formula

    Score = (Visits % × 0.35) + (New Deal Value % × 0.30) + (Revenue % × 0.35)

    If the rep has zero actual visits this month, the score is halved (× 0.5).

    35%
    Visits — check-outs completed vs. visit target
    30%
    New Deal Value — value of deals created vs. new-deal target
    35%
    Revenue — closed-won deal value vs. revenue target

    Badges

    LegendScore ≥ 120
    GoldScore ≥ 90
    SilverScore ≥ 70
    BronzeScore < 70

    Momentum

    ↑ UPScore ≥ 100
    → STEADYScore ≥ 70
    ↓ DOWNScore < 70
